# Paracelsus - A Deno Fresh Template

## Install Paracelsus

```bash
deno install -Ag https://raw.githubusercontent.com/epangelias/paracelsus/refs/heads/main/tasks/paracelsus.ts
```

## Create a new project

```bash
paracelsus my-project
```
